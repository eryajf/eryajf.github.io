之前评论系统使用的gitalk，也不错，新的valine样式更好看一些，于是折腾了很久。

暂时汇总信息如下：

- [Valine官网](https://valine.js.org/)
- [Vuepress-阅读量统计](https://heshiyu1996.github.io/tool/vuepress-stat/#%E5%87%86%E5%A4%87%E5%B7%A5%E4%BD%9C)

- [评论数据存储位置](https://console.leancloud.cn/apps/qfjUPG41m9x43YIhwnnw4EYn-gzGzoHsz/storage/data/Comment?tab=table)
-  目前评论通知还不行
- [Valine 扩展之邮件通知](http://www.zhaojun.im/hexo-valine-admin/)

  -   目前参考此文章配置，还未成功

  -  使用了占位图，头像旋转，自动隐藏占位图等功能，很优秀

```sh
yarn add vuepress-plugin-comment -D
```

### 3，美化

#### 在 veditor 输入界面增加背景图

![comment](http://t.eryajf.net/imgs/2021/09/e64cd5895a5a2423.jpg)

使用方法如下，将 CSS 加到 `docs/.vuepress/styles/index.styl`。

```css
#veditor {
	background-image: url(https://cdn.jsdelivr.net/gh/drew233/cdn/20200409110727.webp);
	background-size: contain;
	background-repeat: no-repeat;
	background-position: right;
	background-color: rgba(255, 255, 255, 0);
	resize: vertical
}
```

上述的 url 内的图片可以换成自己的。如果评论者在输入框中输入较长的内容，那么背景图会遮住部分文字，这样引起整体观感欠佳。设置当鼠标聚焦于输入框时，背景图消失，输入框失去焦点时，背景图重新显示，这个逻辑可以直接交给 CSS 完成。

```css
#veditor:focus{
	background-position-y: 200px;
	transition: all 0.2s ease-in-out 0s;
}
```

#### 将 nick、email、link 输入框分开

这个就根据自己改了。

![输入框分开](https://cdn.jsdelivr.net/gh/zhixuan666/gh-blog@master/images/b17cfc64/box.png)

使用前请确保 `Valine` 评论的 `div` 的 `id` 是 `vcomments`，即 `<div id="vcomments"></div>` 如果是 `class` 为 `vcomments` ，那么将以下的 `#` 换成 `.` 即可。

```css
#vcomments .vheader .vnick {
      width: 31%;

      padding-right: 10px;

#vcomments .vheader .vlink {

```
