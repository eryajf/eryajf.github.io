    # 0 ，前言

    周刊维护在：https://github.com/eryajf/learning-weekly (opens new window) 欢迎投稿，推荐或自荐项目 /文章 /博客，请提交 issue 。

    周刊核心为运维周刊，还会侧重Go语言生态，Vue相关技术生态的项目，以及 GitHub 上优秀项目或经验。

    你也可以在我的博客 https://wiki.eryajf.net/learning-weekly/ (opens new window) 查看汇总周刊。

    # 1，优秀项目

    • 项目地址：halo (opens new window)
    • 项目说明：一款现代化的开源博客 / CMS 系统。这个项目现在已经颇具影响力，文档以及项目配置都是值得学习的，有很深厚的积累。
    • 相关文章：README (opens new window),官网 (opens new window),文档站 (opens new window)，论坛 (opens new window)
    • 项目地址：heyui-admin (opens new window)
    • 项目说明：基于 vue 和 heyui 组件库的中后端系统
    • 相关文章：README (opens new window)
    • 项目地址：dingtalk (opens new window)
    • 项目说明：go版本的钉钉开发api，接口比较全，维护比较稳定
    • 相关文章：README (opens new window)
    • 项目地址：go-redis-manager (opens new window)
    • 项目说明：redis 客户端管理平台(redis manager)【部署简单便捷，SSH连接，用户校验，操作日志等】
    • 相关文章：README (opens new window)
    • 项目地址：github-style (opens new window)
    • 项目说明：一个github风格的hugo博客主题，蛮有意思的。
    • 相关文章：README (opens new window)
    • 项目地址：GitHub-userscripts (opens new window)
    • 项目说明：增益GitHub相关功能的油猴脚本，有很多，可以逐个体验
    • 相关文章：README (opens new window)
    • 项目地址：etcd-manage-server (opens new window)
    • 项目说明：golang+vue开发的etcd管理后台。
    • 相关文章：README (opens new window)
    • 项目地址：etcdWp (opens new window)
    • 项目说明：Macos ETCD UI Desktop（ETCD Mac桌面应用程序）
    • 相关文章：README (opens new window)
