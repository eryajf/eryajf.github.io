Obsidian插件周刊第三期

这篇文章的发布时间较早，其中的内容可能已经过时，阅读时请注意甄别时效性。

# 前言

请参阅：Obsidian插件周刊说明 (opens new window)。

# Quick Switcher++

  • 插件名称：Quick Switcher++
  • 插件作者：darlal
  • 插件说明：增强的快速切换器、搜索打开面板和符号。
  • 插件仓库：点我跳转 (opens new window)

增强版的快速切换与打开窗口，详情参考 (opens new window)。

# Day Planner

  • 插件名称：Day Planner
  • 插件作者：James Lynch
  • 插件说明：一个黑曜石插件，用于从Markdown注释中的任务列表中计划和管理番茄计时器。
  • 插件仓库：点我跳转 (opens new window)

能够添加一天计划的插件，配置页面效果如下：

# Editor Syntax Highlight

  • 插件名称：Editor Syntax Highlight
  • 插件作者：death_au
  • 插件说明：编辑模式下高亮代码块
  • 插件仓库：点我跳转 (opens new window)

编辑模式下高亮代码块，详情参考 (opens new window)。

展示如下：

# Review

  • 插件名称：Review
  • 插件作者：ryanjamurphy
  • 插件说明：在日后的日期(或者过去的日期，你是时间旅行者)的每日笔记中添加一个链接到当前的笔记。
  • 插件仓库：点我跳转 (opens new window)

支持将当前笔记以链接的形式添加到未来某个日期笔记里，从而能够让你在未来看到当前这篇笔记。

官方动图如下：

# Advanced Appearance

  • 插件名称：Advanced Appearance
  • 插件作者：@kepano
  • 插件说明：更改颜色、字体和其他外观设置
  • 插件仓库：点我跳转 (opens new window)

提供了丰富的自定义ob外观的能力，不过感觉这种外观上的折腾要更加谨慎，找到自己满意的主题之后，就不建议折腾太多了。

# Hider

  • 插件名称：Hider
  • 插件作者：@kepano
  • 插件说明：隐藏工具提示、状态、标题栏等UI元素
  • 插件仓库：点我跳转 (opens new window)

如说明介绍所言，详情参考 (opens new window)。

# Minimal Theme Settings

  • 插件名称：Minimal Theme Settings
  • 插件作者：@kepano
  • 插件说明：控制最小主题中的颜色和字体
  • 插件仓库：点我跳转 (opens new window)

这个插件配合Minimal Theme (opens new window)主题使用，如果你用的是这个主题，可以详细了解。

# System Dark Mode

  • 插件名称：System Dark Mode
  • 插件作者：@kepano
  • 插件说明：自动使用操作系统的设置在明暗模式之间切换
  • 插件仓库：点我跳转 (opens new window)

此插件启用之后，主题的明暗模式将根据系统的明暗模式变换。

# Discord Rich Presence

  • 插件名称：Discord Rich Presence
  • 插件作者：Luke Leppan
  • 插件说明：更新您的 Discord 状态，向您的朋友展示您在 Obsidian 中的工作。
  • 插件仓库：点我跳转 (opens new window)

Discord是一个在线互动平台，通常国内用户应该用到不多，因此这个插件大多数同学应该用不到。

# Templater

  • 插件名称：Templater
  • 插件作者：SilentVoid
  • 插件说明：创建和使用模板
  • 插件仓库：点我跳转 (opens new window)

提供了比核心插件中的模板更加强大的能力，详情参考 (opens new window)。
