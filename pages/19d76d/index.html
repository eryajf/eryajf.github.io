    # 0 ，前言

    周刊维护在：https://github.com/eryajf/learning-weekly (opens new window) 欢迎投稿，推荐或自荐项目 /文章 /博客，请提交 issue 。

    周刊核心为运维周刊，还会侧重 Go 语言生态，Vue 相关技术生态的项目，以及 GitHub 上优秀项目或经验。

    你也可以在我的博客 (opens new window) 查看汇总周刊。

    🔥 有不少人想单独从博客通过 RSS 订阅周刊的更新，现在它来了，你可以使用这个🔗 链接 (opens new window)进行订阅。

    # 1，优秀项目

    • 项目地址：logto (opens new window)
    • 项目说明：Logto 帮助你在数分钟内构建登录体验与用户身份体系。
    • 相关文章：官网 (opens new window)
    • 项目地址：fiber (opens new window)
    • 项目说明：用 Go 编写的受 Express 启发的 Web 框架
    • 相关文章：README (opens new window)
    • 项目地址：NotionNext (opens new window)
    • 项目说明：一个使用 NextJS + Notion API 实现的，部署在 Vercel 上的静态博客系统。为 Notion 和所有创作者设计。
    • 相关文章：README (opens new window)
    • 项目地址：open-golang (opens new window)
    • 项目说明：一个能够打开系统默认浏览器的 go 库。比如下边的代码运行之后会打开我的博客：
    	open.Run("https://eryajf.net/")
