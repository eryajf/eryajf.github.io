    # 0 ，前言

    周刊维护在：https://github.com/eryajf/learning-weekly (opens new window) 欢迎投稿，推荐或自荐项目 /文章 /博客，请提交 issue 。

    周刊核心为运维周刊，还会侧重 Go 语言生态，Vue 相关技术生态的项目，以及 GitHub 上优秀项目或经验。

    你也可以在我的博客 (opens new window) 查看汇总周刊。

    🔥 有不少人想单独从博客通过 RSS 订阅周刊的更新，现在它来了，你可以使用这个🔗 链接 (opens new window)进行订阅。

    # 1，优秀项目

    • 项目地址：go-querystring (opens new window)
    • 项目说明：将结构体编码为 URL 查询字符串参数。当然一些 web 框架可能已经集成了这个能力。
    • 相关文章：README (opens new window)
    • 项目地址：npkill (opens new window)
    • 项目说明：该工具允许您列出系统中的任何 node_MODULES 目录，以及它们占用的空间。然后，您可以选择要擦除哪些文件以释放空间。
    • 相关文章：README (opens new window)
    • 项目地址：go-if-expression (opens new window)
    • 项目说明：Golang 中的三元表达式的折中实现，用于减少代码行数提高可读性
    • 相关文章：README (opens new window)
    • 项目地址：go-musicfox (opens new window)
    • 项目说明：go-musicfox 是用 Go 写的又一款网易云音乐命令行客户端，支持 UnblockNeteaseMusic、各种音质级别、lastfm、MPRIS、MacOS 交互响应。
    • 相关文章：README (opens new window)
    • 项目地址：vscode-chatgpt-reborn (opens new window)
    • 项目说明：vscode 中最好用的 chatgpt 插件，原版作者删插件不维护了，这是 fork 出来的一个版本，还很好用。
    • 相关文章：README (opens new window)
    • 项目地址：diskusage (opens new window)
    • 项目说明：一个显示磁盘使用情况的工具。支持跨平台。
    • 相关文章：README (opens new window)
