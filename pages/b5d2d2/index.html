    广告招商

    本周刊公开招商，如有想在周刊投放广告的，请通过 邮件 与我联系。

    # 0 ，前言

    周刊维护在：https://github.com/eryajf/learning-weekly (opens new window) 欢迎投稿，推荐或自荐项目 /文章 /博客，请提交 issue 。

    周刊核心为运维周刊，还会侧重 Go 语言生态，Vue 相关技术生态的项目，以及 GitHub 上优秀项目或经验。

    你也可以在我的博客 (opens new window) 查看汇总周刊。

    🔥 有不少人想单独从博客通过 RSS 订阅周刊的更新，现在它来了，你可以使用这个🔗 链接 (opens new window)进行订阅。

    # 1，优秀项目

    • 项目地址：apinto (opens new window)
    • 项目说明：基于 golang 开发的网关。具有各种插件，可以自行扩展，即插即用。此外，它可以快速帮助企业管理 API 服务，提高 API 服务的稳定性和安全性。
    • 相关文章：README (opens new window)，官网 (opens new window)
    • 项目地址：chroma (opens new window)
    • 项目说明：一个 go 语言库，能够给其他语言高亮的库。
    • 相关文章：README (opens new window), 在线演示 (opens new window)
    • 项目地址：pock (opens new window)
    • 项目说明：一款免费、开源的 MacBook Touch Bar 小部件管理工具。
    • 相关文章：README (opens new window)
    • 项目地址：nginx-ui (opens new window)
    • 项目说明：一个 Nginx 配置管理的 web 系统。
    • 相关文章：README (opens new window)
    • 项目地址：MaxKey (opens new window)
    • 项目说明：Java 语言编写的单点登录认证系统。
    • 相关文章：README (opens new window)

    # 2，优秀文章

    • 【vue3-element-admin】基于 Vue3 + Vite4 + TypeScript + Element-Plus 从 0 到 1 搭建后台管理系统 (opens new window)
    • 如题
    • SSHA Password Hash with Golang (opens new window)
    • 通过这篇文章，能够生成 ldap 中可以识别的 ssha 加密算法的用户密码。
    • OpenLDAP 使用 slappasswd 生成密码 (opens new window)
    • 如题，通过 slappasswd 命令生成明文以及各种加密方法的密码。
