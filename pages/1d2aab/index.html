    周刊维护在：https://github.com/eryajf/learning-weekly (opens new window) 欢迎投稿，推荐或自荐项目 /文章 /博客，请提交 issue 。

    # 1，优秀项目

    • 项目地址：osp (opens new window)
    • 项目说明：OpenSpace是一个开源的可以同时管理多个不同云厂商的Kubernetes集群管理平台。
    • 相关文章：README (opens new window)
    • 项目地址：china_regions (opens new window)
    • 项目说明：最全最新中国省市，地区json及sql数据
    • 相关文章：README (opens new window)
    • 项目地址：OMP (opens new window)
    • 项目说明：OMP是云智慧公司自主设计、研发，为用户提供便捷运维能力和业务管理的轻量级、聚合型、智能运维管理平台。
    • 相关文章：README (opens new window)
    • 项目地址：sonic (opens new window)
    • 项目说明：一个极快的 JSON 序列化和反序列化库
    • 相关文章：README (opens new window)
    • 项目地址：cloudiac (opens new window)
    • 项目说明：CloudIaC 是基于基础设施即代码构建的云环境自动化管理平台。 CloudIaC 将易于使用的界面与强大的治理工具相结合，让您和您团队的成员可以快速轻松的在云中部署和管理环境。
    • 相关文章：官网 (opens new window)，官方文档 (opens new window)
    • 项目地址：hidden (opens new window)
    • 项目说明：Mac电脑中顶部状态栏图标隐藏工具
    • 相关文章：README (opens new window)
    • 项目地址：KubePi (opens new window)
    • 项目说明：KubePi 是一款简单易用的开源 Kubernetes 可视化管理面板
    • 相关文章：官方文档 (opens new window)
    • 项目地址：awesome-tools (opens new window)
    • 项目说明：作者整理了他收集的常用软件。
