    学习周刊-总第13期-2021年第30周

    文章发布较早，内容可能过时，阅读注意甄别。

    # 0 ，前言

    周刊维护在：https://github.com/eryajf/learning-weekly (opens new window) 欢迎投稿，推荐或自荐项目 /文章 /博客，请提交 issue 。

    周刊核心为运维周刊，还会侧重Go语言生态，Vue相关技术生态的项目，以及 GitHub 上优秀项目或经验。

    你也可以在我的博客 https://wiki.eryajf.net/learning-weekly/ (opens new window) 查看汇总周刊。

    # 1，开源项目

    • 项目地址：Whaler (opens new window)
    • 项目说明：能够查看在运行容器的Dockerfile的工具，有时候我们可能会维护一个没有Dockerfile的容器，即可借助此工具。
    • 相关文章：博文介绍 (opens new window)
    • 项目地址：runlike (opens new window)
    • 项目说明：给定一个现有的 docker 容器，打印运行它的副本所需的命令行。
    • 相关文章：README

    • 项目地址：kubecolor (opens new window)

    • 项目说明：可以给kubectl命令行着色的一个工具。

    • 相关文章：README (opens new window)

    # 2，优秀文章

    • 腾讯面试：如何调试Docker容器 (opens new window)
    • 文章介绍了通过几种不太常用但好用的容易调试命令行工具
    • 如何写出优雅的 Go 语言代码 (opens new window)
    • 经验丰富的过来人的最佳实践
    • 如何在kubernetes中优雅停机 (opens new window)
    • 文章介绍了使用脚本管理容器进程而服务进程ID不为1的处理方式，另外可参考的文章：
    • How to propagate SIGTERM to a child process in a Bash script (opens new window)
    • 滴滴pprof定位问题 (opens new window)
    • go语言项目的CPU问题定位思路
    • 天才源自刻意练习 (opens new window)
    • 值得刻意阅读
    • 接入层问题故障定位 (opens new window)
    • 文章介绍了从运维角度针对os层面的问题定位排查思路
    • 程序员找工作指南 (opens new window)
    • 作者从一个基础入门的角度讲解了互联网公司找工作的术语以及套路，很赞

    # 3，优秀博客

    • 博客地址：Kubernetes指南 (opens new window)
    • 简单说明：如题，作者系统地整理了k8s的基础内容，以及一些实践
    • 博客地址：蜂鸣器 (opens new window)
    • 简单说明：作者是一个平面设计师，里边每天更新的图非常有意思
    • 博客地址：阿里云盘资源分享 (opens new window)
    • 简单说明：一个阿里云盘资源分享论坛。
    • 博客地址：William Guo (opens new window)
    • 简单说明：wayne发起者，k8s，golang。

    • 简单说明：k8s，文章有内容，值得读
