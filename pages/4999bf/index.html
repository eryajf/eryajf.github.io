# 笔记

data 与 el 的两种写法：

  1. el 的两种写法
  • new Vue 时配置 el 属性。
  • 先创建 Vue 实例，然后再通过 vm.$mount(#root) 的方式进行挂载 2. data 的两种写法 - 对象式 - 函数式

如何选择：目前学习阶段都可以，后边学习到组件时，data 必须使用函数式，否则会出问题。 3. 一个重要原则 由 Vue 管理的函数，一定不要写箭头函数，一旦写了箭头函数，this 就不再是 Vue 实例了。
