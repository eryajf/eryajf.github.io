    # 0 ，前言

    周刊维护在：https://github.com/eryajf/learning-weekly (opens new window) 欢迎投稿，推荐或自荐项目 /文章 /博客，请提交 issue 。

    周刊核心为运维周刊，还会侧重Go语言生态，Vue相关技术生态的项目，以及 GitHub 上优秀项目或经验。

    你也可以在我的博客 https://wiki.eryajf.net/learning-weekly/ (opens new window) 查看汇总周刊。

    # 1，优秀项目

    • 项目地址：refined-github (opens new window)
    • 项目说明：一个浏览器扩展，针对于github浏览体验增强的插件。
    • 相关文章：README (opens new window)
    • 项目地址：git-history (opens new window)
    • 项目说明：从任何git存储库中快速浏览文件的历史记录
    • 相关文章：README (opens new window)，官网 (opens new window)
    • 项目地址：github-i18n-plugin (opens new window)
    • 项目说明：GitHub汉化油猴插件，效果不错。
    • 相关文章：README (opens new window)
    • 项目地址：picx (opens new window)
    • 项目说明：一款基于 GitHub API & jsDelivr 开发的具有 CDN 加速功能的图床管理工具
    • 相关文章：README (opens new window)
    • 项目地址：readme.so (opens new window)
    • 项目说明：一款能够在线制定项目README的工具。
    • 相关文章：官网 (opens new window)
    • 项目地址：qmgo (opens new window)
    • 项目说明：Qmgo 是一款Go语言的MongoDB driver，它基于MongoDB 官方 driver (opens new window) 开发实现，同时使用更易用的接口设计，比如参考mgo (opens new window) （比如mgo的链式调用）。
    • 相关文章：README (opens new window)
    • 项目地址：ebpfmanager (opens new window)
    • 项目说明：ebpfmanager参照datadog/ebpf/manager包的思想，基于cilium/ebpf实现的ebpf类库封装。
    • 相关文章：README (opens new window)
    • 项目地址：mizu (opens new window)
    • 项目说明：适用于Kubernetes的API流量查看器，使您能够查看微服务之间的所有API通信。想想TCPDump和Wireshark是为Kubernetes重新发明的。
    • 相关文章：官网 (opens new window)
    • 项目地址：PPTist (opens new window)
    • 项目说明：基于Vue实现的在线PPT应用。
    • 相关文章：README (opens new window)，在线PPT (opens new window)
